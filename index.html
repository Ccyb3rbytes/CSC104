HTML Elements

An HTML element represents a part of a webpage, such as a paragraph, heading, link, or image. It defines the structure and content of the webpage using opening and closing tags (e.g., &1t;;p>&1t;/p> for a paragraph).
Category Elements Description
Document Structure <!DOCTYPE>, <html>, <head>, <title>, <body> Defines the basic structure of an HTML document.
Text Formatting <p>, <br>, <hr>, <b>, <cite>, <code>, <em>, <i>, <strong>, <u> Formats and styles text content.
Images <img>, <figure>, <figcaption> Embeds and captions images or self-contained media.
Lists <ul>, <ol>, <li> Creates ordered and unordered lists.
Tables <table>, <th>, <tr>, <td> Creates tables, headers, rows, and cells.
Layout <div>, <span>, <header>, <footer>, <main>, <section>, <article>, <aside> Structures page layout and sections.

HTML Attributes

An HTML Attribute provides additional information about an HTML element. Attributes are included inside the opening tag and usually come in name-value pairs ie: (class="example") helping define properties like ID's classes, or links
Usage Description
<div id="header">, <p class="intro"> Specifies a unique ID or class for styling or JavaScript targeting.
<img src="image.jpg" alt="description"> Specifies the source of an image and alternative text for accessibility.
<a href="https://example.com"> Specifies the URL of a link destination.
<p style="color: red;"> Specifies inline CSS styles directly in an HTML element.
<p title="Hello"> Provides extra information about an element (appears as a tooltip).
<a href="https://example.com" target="_blank"> Specifies where to open a linked document (e.g., new tab).
<input type="text" placeholder="Enter your name"> Specifies a short hint displayed in an input field.
<button disabled>Submit</button> Disables an element, making it unclickable.

CSS Properties

CSS properties control the appearance and layout of HTML elements. They define characteristics like color, font size, margin, padding, and border styles, allowing you to customize how a webpage looks.
Usage Description
color: red; Sets the color of text.
background-color: blue; Sets the background color of an element.
font-size: 16px; Sets the size of the font.
margin: 20px; Sets the margin area outside an element.
padding: 10px; Sets the padding area inside an element.
border: 1px solid black; Sets the border around an element.
text-align: center; Sets the horizontal alignment of text.
display: block; Specifies the display behavior of an element.
position: absolute; Sets the positioning method for an element.
opacity: 0.5; Sets the transparency level of an element.

CSS Selectors

CSS selectors are patterns used to target and style specific HTML elements. They can select elements by name, class, ID, or their relationship to other elements, making it easier to apply consistent styling across a webpage.
Category Description
Simple Selectors Select elements based on name, id, or class.
Combinator Selectors Select elements based on a specific relationship between them.
Pseudo-class Selectors Select elements based on a certain state.
Pseudo-elements Select and style a part of an element.
